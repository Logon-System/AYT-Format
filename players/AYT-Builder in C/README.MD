#AYT PC BUILDER
-------------------------

Dans cette section, vous trouverez les sources et exécutables qui permettent de remplacer le constructeur du player AYT par un programme en C (Ayt-Builder)

Ce programme permet de faire le même travail que AYT-Builder et générer un binaire en Z80A et un fichier de données AYT initialisé, mais via un exécutable en ligne de commande.

L'objectif est de permettre d'intégrer le Builder AYT dans une chaine de compilation complète sans avoir besoin de recourir à une exécution Z80A sur CPC.

(cela reste toutefois possible avec des solutions d'émulation du Builder (via un émulateur avec script) ou des solutions d'assemblage avec émulation Z80A associée).

Les sources, exécutables, musiques, binaires, résultats de tests sont documentés dans la section de la plateforme correspondante.

Le Builder-Cest actuellement disponible uniquement pour la plateforme "CPC classique" (ancienne génération) pour un player utilisé en RAM.
