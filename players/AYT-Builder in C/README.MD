#AYT PC BUILDER
-------------------------

In this section, you will find the sources and executables that replace the AYT player builder with a program written in C (Ayt-Builder).

This program performs the same task as **AYT-Builder** and generates a **Z80A binary** as well as an **initialized AYT data file**, but via a **command-line executable**.

The goal is to allow the integration of the AYT Builder into a complete cross-development toolchain without the need for executing Z80A code on a CPC.

(This, however, remains possible using Builder emulation solutions (via a scripted emulator) or assembly solutions combined with Z80A emulation).

The sources, executables, music, binaries, and test results are documented in the corresponding platform section.

The C-Builder is currently available only for the "classic CPC" platform (old generation) for a player running in RAM.


#AYT PC BUILDER
-------------------------

Dans cette section, vous trouverez les sources et exécutables qui permettent de remplacer le constructeur du player AYT par un programme en C (Ayt-Builder)

Ce programme permet de faire le même travail que **AYT-Builder** et générer un **binaire en Z80A** et un **fichier de données AYT initialisé**, mais via un **exécutable en ligne de commande**.

L'objectif est de permettre d'intégrer le Builder AYT dans une chaine de compilation crossdev complète sans avoir besoin de recourir à une exécution Z80A sur CPC.

(cela reste toutefois possible avec des solutions d'émulation du Builder (via un émulateur avec script) ou des solutions d'assemblage avec émulation Z80A associée).

Les sources, exécutables, musiques, binaires, résultats de tests sont documentés dans la section de la plateforme correspondante.

Le Builder-Cest actuellement disponible uniquement pour la plateforme "CPC classique" (ancienne génération) pour un player utilisé en RAM.
